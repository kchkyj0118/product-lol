# AI 롤 승리 코치 (AI LOL Victory Coach)

## 1. 개요 (Overview)
이 프로젝트는 리그 오브 레전드(LoL)의 실시간 챔피언 조합과 라인 정보를 바탕으로 AI가 최적의 승리 전략을 분석해주는 시뮬레이션 웹 애플리케이션입니다. 닉네임 조회 방식이 아닌, 사용자가 직접 챔피언을 선택하여 상성과 운영법을 분석받을 수 있는 전략 시뮬레이터입니다.

## 2. 주요 기능 및 디자인 (Features & Design)
- **전략 시뮬레이션:** 아군 5명, 적군 5명의 챔피언을 직접 선택.
- **포지션 선택:** 사용자의 주 라인(탑, 정글, 미드, 원딜, 서포터) 지정.
- **AI 분석:** Cloudflare Pages Functions와 Gemini API를 활용한 맞춤형 승리 플랜 생성.
- **디자인:** LoL 감성을 담은 다크 모드 UI, 세련된 애니메이션 및 반응형 레이아웃.
- **결과 구성:** 초반(라인전/초반 교전), 중반(오브젝트/한타), 후반(사이드/최종 한타) 단계별 가이드 제공.

## 3. 구현 상세 (Implementation Details)

### 백엔드 (`functions/api/analyze.js`)
- **기술 스택:** Cloudflare Pages Functions
- **기능:** 
  - 프론트엔드로부터 아군/적군 챔피언(10명) 및 사용자 라인 정보 수신.
  - Gemini 1.5 Flash API를 호출하여 전문 코치 컨셉의 분석 데이터 생성.
  - 결과를 '초반', '중반', '후반'으로 구분하여 반환.

### 프론트엔드 UI (`public/index.html`, `public/style.css`)
- **디자인 컨셉:** League of Legends 클라이언트 감성의 다크 모드.
- **주요 요소:**
  - 챔피언 선택 슬롯 (아군/적군 각 5개).
  - 챔피언 검색 및 선택 모달.
  - 라인 선택 버튼 그룹.
  - 분석 결과 표시 카드 (단계별 섹션 구분).

### 프론트엔드 로직 (`public/app.js`)
- **챔피언 데이터:** 주요 챔피언 리스트 내장 및 필터링 검색 기능.
- **상태 관리:** 선택된 챔피언 및 라인 상태 추적.
- **API 연동:** fetch API를 통한 백엔드 통신 및 결과 파싱 렌더링.
