# lol-coaches - Project Blueprint

## Overview
**lol-coaches**는 Riot Games API와 Google Gemini AI를 결합하여 리그 오브 레전드(LoL) 유저에게 실시간 인게임 전략 분석 및 승리 공식을 제공하는 차세대 AI 코칭 플랫폼입니다.

## Project Vision & Goals
- **Real-time Insights:** 사용자가 게임 로딩 중에도 즉각적인 상성 분석과 운영 전략을 얻을 수 있도록 함.
- **Data-Driven Strategy:** 단순 통계를 넘어 AI가 정밀 분석한 '승리 시나리오' 제공.
- **Educational Value:** 유저의 게임 이해도를 높여 장기적인 실력 향상을 도모함.

## Detailed Development Plan (AdSense Approval Focus)

### 1. High-Value Content Strategy
- **Index Page:** 
    - 실시간 분석 기능 외에도 '오늘의 AI 추천 메타', '라인전 상성 가이드' 등 정적 콘텐츠를 강화하여 첫 방문자에게 즉각적인 가치 제공.
    - "ADVERTISEMENT SPACE"와 같은 미완성 요소를 제거하고 실제 유용한 정보 섹션으로 대체.
- **Guide Page (The "Knowledge Base"):**
    - 단순 매뉴얼을 넘어선 전문적인 LoL 공략 아카이브 구축.
    - 예: "AI가 분석한 2026 시즌 메타 변화", "데이터로 증명된 초반 주도권의 중요성" 등 고품질 아티클 게시.
- **About Page:**
    - 서비스의 기술적 아키텍처(Gemini API, Riot API 연동 방식)와 팀의 전문성을 강조하여 신뢰도 확보.

### 2. Design & UX Excellence
- **Modern Aesthetic:** 다크 모드 기반의 고대비 디자인, 네온 블루 포인트를 활용한 프리미엄 테크 느낌 구현.
- **Interactive Feedback:** 분석 로딩 중에도 사용자에게 유용한 팁을 노출하여 이탈 방지.
- **Mobile Responsive:** 모든 기기에서 완벽한 가독성 확보.

### 3. Compliance & Standards
- **AdSense Policy Alignment:** 
    - 가치 없는 콘텐츠 제거 (No Low Value Content).
    - 사용자 경험 방해 요소 제거 (No Distracting Navigation).
    - 필수 정책 페이지(Privacy, Terms) 최신화 및 명확한 링크 배치.

## Technical Stack
- **Frontend:** Vanilla JS, Tailwind CSS (Modern Baseline Features).
- **Backend:** Firebase Functions (Node.js).
- **AI:** Google Gemini AI (Flash Preview).
- **Data:** Riot Games Spectator & Match V5 API.

## Implementation Steps (Current Phase)
1. **[Current]** `index.html` 내 미완성 요소 제거 및 가치 있는 정보 섹션 추가.
2. `guide.html`을 전문적인 공략 아티클 위주로 전면 개편.
3. `about.html`에 기술적 디테일 및 전문성 강화 콘텐츠 추가.
4. 전체 사이트 검수 및 더미 텍스트 유무 확인.
5. GitHub 메인 브랜치 푸시 및 자동 배포 확인.
