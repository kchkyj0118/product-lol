How to use

01. 소환사 정보 입력

메인 페이지의 검색창에 분석하고 싶은 소환사명과 태그(#KR1 등)를 입력하세요. 현재 게임 중인 데이터(실시간 인게임 정보)를 기반으로 분석이 진행됩니다.

02. 실시간 데이터 로딩

'전략 분석' 버튼을 클릭하면 AI가 라이엇 API를 통해 현재 팀 구성, 챔피언 상성, 핵심 룬 데이터를 수집합니다. 분석에는 약 3~5초 정도 소요됩니다.

03. 1타 강사의 필승 전략 확인

분석이 완료되면 ⚡ Winning Formula 섹션에서 요약된 핵심 전략을 확인할 수 있습니다. 'Detail Analysis +' 버튼을 눌러 포지션별 상세 운영법과 한타 포지셔닝 팁을 읽어보세요.

04. 승리 쟁취

AI가 제시한 상성 데이터와 3레벨 타이밍, 핵심 동선을 숙지하고 게임에 임하세요. 데이터 기반의 판단이 당신의 승률을 획기적으로 높여줄 것입니다.
