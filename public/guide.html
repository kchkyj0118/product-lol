사용법 가이드

1. 내 라인 선택

가장 먼저 화면 상단의 버튼을 통해 본인이 플레이할 라인을 선택하세요. AI가 해당 포지션에 특화된 1:1 상성과 운영법을 우선적으로 분석합니다.

2. 팀 구성 입력

아군과 적군의 챔피언 이름을 입력하세요. 초성 검색(예: 'ㄱㄹ' -> 가렌)을 지원하며, 챔피언을 선택하면 아이콘이 표시됩니다. 소환사 주문(스펠)까지 정확히 입력하면 더욱 정밀한 분석이 가능합니다.

3. 분석 리포트 확인

'승리 플랜 분석 시작' 버튼을 누르면 AI가 통계 데이터를 계산합니다. 핵심 요약 카드를 먼저 확인하고, '상세 분석' 버튼을 눌러 구체적인 룬 세팅과 아이템 빌드를 확인하세요.

4. 정글러 팁

정글러를 선택하신 경우, 분석 결과 하단에 미니맵 동선이 시각화됩니다. AI가 추천하는 최적의 동선을 따라 초반 주도권을 가져오세요.
